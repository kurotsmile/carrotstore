# carrotstore
Dự án kho lưu trữ các ứng dụng và các dịch vụ số giải trí

## Cách thiết lập sau khi clone về:
Thiết lập lại file `config.php` ở localhost

## Cách chức năng chính:
* Ứng dụng và trò chơi
* Danh bạ trực tuyến
* Kho nhạc trực tuyến
* Kho châm ngôn
* Piano
